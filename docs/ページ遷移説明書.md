## 📌 使い方（PlantUML 表示）
1. VS Code の PlantUML 拡張を使うと編集 → プレビューができます。
2. CLI で表示する場合（ローカルに PlantUML を用意している場合）:
   - 例 (PowerShell):
     ```powershell
     plantuml docs/navigation_pre_auth.puml
     plantuml docs/navigation_post_auth.puml
     ```
   - これで `*.png` や `*.svg` が生成されます（環境依存）。
