# /home のバックアップについて

このリポジトリでは、認証後の `
/home` ページからアクセスできるシンプルなバックアップとして **/home2** と **/home3** を用意しています。

目的:
- 実験用のトップページを分離しておくことで、UI やメニューの変更を安全に試せるようにする

現状:
- `/home2` — 左サイドメニューを使ったサンプルワークスペース（`src/app/home2`）
- `/home3` — shadcn/ui で作成した認証後トップページ（`src/app/home3`）

使用方法:
- 開発中は `/(protected)/(menu)/(1-home)/home` のトップから `/home2` や `/home3` へ移動して挙動を確認できます。

メンテナンス:
- これらは「バックアップ兼実験ワークスペース」として継続的に維持します。必要な時は同名でコピーを取り、実験のたびに変更履歴を残してください。
