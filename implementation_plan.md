## TDD とコロケーション方針（必須）🧪

- 全ての新規 UI コンポーネント / ビジネスロジックはテストファーストで作成
- RED → GREEN → REFACTOR サイクルを厳守
- コンポーネント単位で近くにテストファイルを配置
  - 例: `Component.tsx` と同ディレクトリに `Component.test.tsx`
